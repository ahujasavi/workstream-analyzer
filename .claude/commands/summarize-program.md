# Summarize Program

Run `python3 summary.py` from the project root and show the full output.

Then write a short plain-English paragraph (3-5 sentences) that explains:
- How many workstreams are Green, Amber, and Red
- Which ones need the most urgent attention and why
- The overall health of the program in plain language a non-technical stakeholder could understand

Keep the tone calm and factual — no bullet points in the paragraph, just prose.
