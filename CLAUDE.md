# Workstream Analyzer

A small Python project that reads a CSV of program workstreams and prints a summary grouped by status.

## Files

- `workstreams.csv` — the data file with 6 workstreams
- `summary.py` — the script that reads the CSV and prints the summary

## How to run

```bash
python summary.py
```
